# Aspectus Frontend

Этот репозиторий содержит фронтенд для приложения с ИИ формированием черновика отчета для онкологический исследований 

## Структура проекта

- `package.json`: Описание проекта, его зависимости и скрипты для сборки/разработки.
- `public/`: Каталог со статическими файлами.
- `index.html`: Основной HTML файл, в который встраивается Vue.js приложение.
- `src/`: Основной каталог с исходным кодом приложения.
  - `App.jsx`: Главный компонент приложения.
  - `assets/`: Каталог для хранения ресурсов, таких как изображения и стили.
  - `components/`: Каталог с React компонентами.
    - `ButtonClick.jsx`: Компонент для создания кнопок.
    - `InputField.jsx`: Компонент для создания полей ввода
  - `pages/`: Каталог с React страницами
    - `MainForm.jsx`: Страница с формой для заполнения данных
    - `NotFound.jsx`: Страница для вывода 404 ошибки
  - `main.jsx`: Точка входа для React.js приложения.
  - `styles.css`: Основные стили для приложения.

- `vite.config.js`: Конфигурация для сборки проекта с использованием Vite.

## Установка зависимостей

Перед началом работы с проектом необходимо установить все необходимые зависимости. Для этого выполните следующую команду:

```bash
npm install
```

## Сборка и запуск

### Режим разработки

Для запуска приложения в режиме разработки с горячей перезагрузкой используйте команду:

```bash
npm run dev
```

### Сборка для продакшена

Чтобы сделать сборки и запустить приложение для продакшена выполните команду:

```bash
npm run deploy
```
